# 🎥 Netflix Clone

A feature-packed Netflix Clone app that displays movies and shows in different categories with both horizontal and vertical scrolling lists, video trailer playback, search, offline caching, and user profile management — all in a clean, modern interface.

---

## 📱 Features

* **Vertical scrolling list of categories**
* **Horizontal scrolling lists of movies within each category**
* Movie detail screen with title, image, and description
* **Video trailer playback**
* **Search functionality to find movies and shows**
* **Offline data caching**
* **User login and multiple profiles**
* Clean, Netflix-inspired UI
* MVVM architecture
* Dark Mode support

---

## 🛠️ Tech Used

* Swift
* UIKit
* MVVM Architecture
* URLSession
* Kingfisher for image caching
* Core Data for offline caching
* GDC for user login

## 📦 How to Run

1. Clone the project:

   ```bash
   git clone https://github.com/your-username/netflix-clone.git
   ```

2. Open the project in Xcode:

   ```bash
   xed .
   ```

3. Build and run on the iOS Simulator or a physical device.
